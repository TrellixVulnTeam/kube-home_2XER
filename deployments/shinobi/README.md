# Shinobi

### Install with:

```
cd deployments/shinobi
./install.sh
```

Must navigate to `http://127.0.0.1/super` and create user
before being able to use service.

#### Passwords

Please note, root password is generated randomly with each install

### Uninstall with:

```
cd deployments/zoneminder
./uninstall.sh
```
